# Website Page Guides and Shortcodes

v1.4.6 updates the Tranter Engine backend so the Website area becomes the reference point for page setup.

## What changed

- Website and Website Structure guidance are consolidated into the Website workspace.
- Website Sections is renamed to Section Manager to avoid confusion.
- Website now includes Page Guides with recommended shortcode order for Homepage, Insights, Company, Services, Resources and Campaigns.
- Website now includes a Shortcodes Reference with status labels: Available, Planned, Deprecated.
- Header and footer shortcodes are included in recommended page flows.

## Rule moving forward

Every sprint must update:

1. Backend functionality.
2. Frontend shortcodes.
3. Website > Page Guides.
4. Website > Shortcodes Reference.

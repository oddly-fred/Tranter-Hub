# Header v3.0 Review Package

This branch was created for the Tranter Hub Header v3.0 review.

The full downloadable review ZIP has been generated in ChatGPT as `tranter-header-v3.0-review.zip` and includes:

- `tranter-header-v3.0.html`
- `README.md`
- `header-v3.0-config-summary.json`

## Scope implemented

- GeoIP-aware architecture with a single configurable region object.
- Nigeria navigation experience: all services, all partners, Smart Solutions, ManageEngine, Sophos and Zoho.
- Global navigation experience: Smart Solutions, ManageEngine, Sophos and other partners hidden; Zoho CRM, Zoho Desk, Zoho Books and Zoho Workplace shown.
- Sticky header behaviour that shrinks on scroll/hover, gains rounded corners and keeps top spacing on desktop/mobile.
- Interactive mega menu enhancements and keyboard accessibility improvements.
- Analytics hooks for menu, service, partner, demo, WhatsApp, search, AI, CTA and scroll-depth events.
- Compatibility hooks for GA4/GTM, Microsoft Clarity, Meta Pixel, LinkedIn Insight Tag and custom webhook.
- AI Assistant open hook and search-ready architecture.
- Modular engines: GeoIP, Navigation, Analytics, Partner, Service, Search, AI, CTA and Mobile Navigation.

For review, use the downloadable ZIP generated in ChatGPT. After approval, the full header file can be committed into the preferred production path in this repository.

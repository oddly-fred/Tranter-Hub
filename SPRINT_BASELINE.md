# Tranter Hub Sprint Baseline

This branch resets Tranter Hub development back to the last approved plugin foundation supplied as:

`tranter-engine-v1.2.0(1).zip`

## Decision

The previously generated modular scaffold is not the approved foundation because it changed the admin dashboard and removed the richer Tranter Engine workspace experience.

Going forward, all Tranter Hub sprints must build on top of the approved Tranter Engine v1.2.0 package.

## Approved foundation includes

- Mission Control style admin dashboard
- Website Workspace
- Sections editor
- Services
- Partners
- CTAs
- Market rules
- Dashboard 2.0 styling
- Mulish font styling
- Homepage shortcode system
- Existing seeder/content structure
- Existing Insights assets and module structure

## Sprint direction

Future work should extend this foundation with:

1. Improved Insights module
2. Newsletter lead generation
3. Resources/downloads
4. Events
5. Analytics dashboard
6. Sales hub functionality

## Rule

Do not replace the existing dashboard or admin experience with a fresh scaffold. Extend the approved plugin structure instead.
